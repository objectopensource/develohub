Why you should consider Linux

Linux is an operating system kernel (the kernel is a computer program at the core of a computer's operating system with complete control over everything in the system. It is an integral part of any operating system. It is the "portion of the operating system code that is always resident in memory"). It is one of the least used desktop operating systems right now. But if it's something to consider, why do people use it less? Why should you consider it? Find out here!

1. Security

Linux is more immune to viruses mainly because it's not widely used. What that means is, viruses and hackers target Windows because it's used a lot more and there's more chance of the virus "hitting their target". But this doesn't mean Windows is insecure. Microsoft is doing their best to provide good security. Here's an analogy: You have to shoot one of two targets. One is a lot bigger and one is a lot smaller. Whichever you shoot, you get the same result. So, you would rather shoot the bigger target, because there's more chance of hitting it. That's what viruses and hackers would choose, too. Linux is a small target, while Windows is the bigger one.

2. Performance

Linux works on almost anything! Linux is faster because it usually has a lot lesser/smaller background services than Windows. Background services are the programs that don't have any content that is supposed to be shown to you. Some of them are requiered for the system to work, but some are not.

3. Variety

Linux has a large variety of "distros", which are actually just operating systems with the Linux kernel. So, you can choose whichever distro you like.

Linux also has a variety of Desktop Environments, which have different features and look-and-feels.

There are more advantages, but the bigger ones are covered here. Now let's see why Linux isn't popular.

Why is Linux Used Lesser?

Linux is definitely something to consider, but it isn't popular. This is because it just hasn't gained momentum yet, and also because computers usually come pre-installed with Windows and people don't want to take the effort to switch over.

The only disadvantage (in my opinion)

The only problem about Linux (in my opinion!) is the compatibility. This means that some Windows files won't work in Linux, and also that Linux doesn't have many applications.
